HTML Template Tag

The template tag in HTML acts as a mechanism for adding html markup to a page that, while it doesn't render immediately, can be cloned and rendered on the page later via javascript. HTML templates mimic the kind of templating previously done with libraries like handlebars or mustache.

How does it work?

In your html, include a template tag with a way to access the tag via javascript, an id is easiest.

<template id="personal">Apply for a personal loan.</template> <template id="business">Apply for a business loan.</template>

Example

An example of using templates would be an instance where you have choices for some sort of long-form content.

Say you have an application where you have two different registration forms, one for an individual and one for a business. You can use templates to show the correct form depending on the customer's choice.
